Official Go SDK for the [Islo](https://islo.dev) sandbox platform.

## Install

```bash
go get github.com/islo-labs/go-sdk
```

## Usage

Pass an Islo API key (or set `ISLO_API_KEY`) and `islo.New` returns a client that automatically exchanges the key for a session JWT and refreshes it before expiry.

```go
package main

import (
    "context"
    "fmt"

    api "github.com/islo-labs/go-sdk"
    "github.com/islo-labs/go-sdk/islo"
)

func main() {
    c, err := islo.New(islo.WithAPIKey("ak_..."))
    if err != nil {
        panic(err)
    }
    ctx := context.Background()

    sb, err := c.Sandboxes.CreateSandbox(ctx, &api.SandboxCreate{
        Name:  api.String("my-sandbox"),
        Image: api.String("python:3.12-slim"),
    })
    if err != nil {
        panic(err)
    }

    res, err := c.Sandboxes.ExecInSandbox(ctx, &api.ExecInSandboxRequest{
        SandboxName: sb.Name,
        Body: &api.ExecRequest{
            Command: []string{"echo", "Hello from sandbox"},
        },
    })
    if err != nil {
        panic(err)
    }
    fmt.Println("exec:", res.ExecID, "status:", res.Status)

    _, _ = c.Sandboxes.DeleteSandbox(ctx, &api.DeleteSandboxRequest{
        SandboxName: sb.Name,
    })
}
```

The two imports split cleanly: `islo` is the hand-written constructor + auth wrapper; `api` (root package, aliased) holds every generated request/response type.

### Configuration

| Option | Description | Default |
|---|---|---|
| `islo.WithAPIKey(key)` | Islo API key. | `$ISLO_API_KEY` |
| `islo.WithBaseURL(url)` | API base URL. | `$ISLO_BASE_URL` or `https://api.islo.dev` |
| `islo.WithRefreshMargin(d)` | How long before expiry to refresh the JWT. | `60s` |
| `islo.WithHTTPClient(c)` | Bring your own `*http.Client`. Its `Transport` is wrapped for auth; `Timeout`, `Jar`, `CheckRedirect` are preserved. | `&http.Client{}` |

The token cache is keyed on `(baseURL, apiKey)`, so multiple `islo.New` calls with the same API key share one cached token.

## Errors

API errors implement the standard `errors.Is`/`errors.As` interfaces:

```go
import core "github.com/islo-labs/go-sdk/core"

_, err := c.Sandboxes.CreateSandbox(ctx, ...)
if err != nil {
    var apiError *core.APIError
    if errors.As(err, &apiError) {
        // inspect apiError.StatusCode, apiError.Body, etc.
    }
    return err
}
```

## Advanced

### Retries

Requests retry with exponential backoff on `408`, `429`, and `5xx` (default: 2 attempts). Override per-client or per-call:

```go
import option "github.com/islo-labs/go-sdk/option"

c, _ := islo.New(islo.WithAPIKey("ak_..."))
_, err := c.Sandboxes.CreateSandbox(ctx, req, option.WithMaxAttempts(1))
```

### Timeouts

Use a context-based timeout per request:

```go
ctx, cancel := context.WithTimeout(ctx, 5*time.Second)
defer cancel()
_, err := c.Sandboxes.CreateSandbox(ctx, req)
```

### Using the generated client directly

If you already have a session JWT (e.g., issued by your own auth flow), bypass `islo.New`:

```go
import (
    client "github.com/islo-labs/go-sdk/client"
    option "github.com/islo-labs/go-sdk/option"
)

c := client.NewClient(option.WithAPIKey("<session-jwt>"))
```
